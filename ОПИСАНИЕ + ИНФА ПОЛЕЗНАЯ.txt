В процессе верстки использовались семантические html5 теги, технология flexbox и media-запросы. Страницы сайтов адаптивны под смартфоны и планшеты. Сайт корректно работает в браузерах: Google Chrome, Mozilla FireFox, Opera, а так же в браузере Internet Explorer 11.

Модуль PHP: 7.1
MySQL: 8.1


Normalize.css — это небольшой CSS-файл, который обеспечивает для HTML-элементов лучшую кроссбраузерность в стилях по умолчанию. Это современная, готовая к HTML5 альтернатива традиционному reset.css.
(Исправление ошибок стилей в некоторых браузерах таких как Safari или IE)

Google fonts - сервис предоставляющий набор шрифтов.

Font awesome - сервис предоставляющий набор иконок.

JQuery-3.3.1 - библиотека для языка программирования Java Script, используется для упрощения в написание кода. А так же для корректной работы различных плагинов например WOW JS.

Animate css - набор анимаций для сайта.

Wow JS (jQuery плагин) -  позволяет отложить запуск анимации элементов до момента их отображения на экране при прокрутке страницы. Разработчик WOW.js рекомендует использовать Animate.css для анимации. Эта библиотека поддерживает более 70 эффектов.

Папка git + README.MD - для системы контроля версии Git, выгрузка/загрузка с сервера GitHub.

Папка css - хранит таблицы стилей.

Папка database - последний бэкап базы данных.

Папка files - картинки для товаров на сайте.

Папка img - картинки используемые в верстке.

Папка js - для файлов с расширением js

Папка php - файл соединения с БД
